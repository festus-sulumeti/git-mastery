Git Installation Guide

Overview

Whether you are on macOS, Windows, or Linux, it is important to first check if Git is installed on your system before proceeding with the installation process.

By checking if it is installed in your machine will determine if you would like to reinstall, upgrade it, or stick, to the available version that your machine will state. This is especially the case for macOS/unix and/or Linux

Windows

To check if Git is installed, open the command prompt (cmd) and type:

git --version

If Git is not found, follow these steps to install it:

  1. Download the Git installer from the official Git website.
  2. Run the installer and follow the on-screen instructions. You can keep the default options or customize them if needed.
  3. After installation, open Git Bash and verify the installation with:
    4. git --version

    If you see the Git version, the installation was successful.

macOS

To check if Git is installed, open the Terminal and type:

git --version

If Git is not installed, you can install it using Homebrew or Xcode command line tools:

1. Install Git using Homebrew

  1. If Homebrew is not installed, install it by running the following command in Terminal:
    2. /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Once Homebrew is installed, install Git by running:
    3. brew install git
  3. Verify the installation with:
    4. git --version

2. Install Git using Xcode command line tools

  1. Run the following command to install Xcode command line tools:
    2. xcode-select --install

    • Linux

    To check if Git is installed, open the terminal and type:

    git --version

    If Git is not installed, follow the instructions for your specific Linux distribution:

    Ubuntu/Debian-based Systems

    1. Update your package list by running:
      2. sudo apt update
    2. Install Git by running:
      3. sudo apt install git
    3. Verify the installation with:
      4. git --version

    Fedora-based Systems

    1. Install Git by running the following command:
      2. sudo dnf install git
    2. Verify the installation with:
      3. git --version

    Arch-based Systems

    1. Install Git by running the following command:
      2. sudo pacman -S git
    2. Verify the installation with:
      3. git --version
